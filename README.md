# Homebrew Tap for VibeDiff

This tap provides a formula for installing [VibeDiff](https://github.com/malvex/vibediff), a local Git diff viewer with real-time updates and code review features.

## Installation

```bash
brew tap malvex/tap
brew install vibediff
```

Or install directly:

```bash
brew install malvex/tap/vibediff
```

## Available Formulae

- `vibediff` - Local Git diff viewer with real-time updates

## Documentation

For more information about VibeDiff, see the [main repository](https://github.com/malvex/vibediff).
